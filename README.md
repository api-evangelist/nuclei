# Nuclei (nuclei)
Nuclei is an open source vulnerability scanner from ProjectDiscovery that uses YAML-based templates to find security issues in APIs, web apps, and infrastructure. The ProjectDiscovery Cloud Platform also exposes a REST API for managing templates, scans, vulnerabilities, leaks, asset discovery, exports, and more.
